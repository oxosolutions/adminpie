IRIS

There are three interfaces in Integrated Road Traffic Injury Surveillance Platform(IRIS) Project.

Public Website: There is a public website athttp://rti.fhts.ac.in/ which is openly accessible on internet and provides general information about the project.

Admin Portal(Web Interface): Dynamic web application to manage surveillance questions, Providing API to android Application, Accept the responses synchronized by android application,View Stats, Manage Users, View Response data, Export Response Data, Filter Data, Reporting, etc .The Admin Panel can be accessed at following URL (Credentials are provided below) http://iris.fhts.ac.in/

Android Application: The android application can be installed on android operating system based mobiles and tablets.

User roles

There are three user roles to manage and use the web as well as android application.
  1. Admins
  2. Managers
  3. Operators

After using the credentials you will be redirected to the following screen (dashboard). By click on ‘Start Survey’ you can choose a survey type which you want to conduct .
After choose the survey then user will redirect just like following screen
a)On the top of the screen you will see (Question 1/20) . this means in this survey there are (20) question and (1) shows you are currently on question no.4.This number will automatically increase according to the question .

b) on the screen you can see ( How often do you conduct surveys ) . This is a place where the survey question lands and user can read .

c) on the next step user will see the answers as in the form of checkbox, radio buttons ,text fields , date picker , textarea , number etc . These fields will shown as per the survey answer type.

d)Now on the next step you will see three buttons ( Previous , Stop , Next). As the name suggested the next button will land the next question . Previous button will bring the previous question back now when the user click on the previous button then user is able to change the previous question's answer. And then able to go on next question with the click of next button.
At the end of the survey user will get a Thank you note ( when the survey ends
Now in the sidebar you can see a menu bar in which there are seven options
  1. Dashboard
  2. Survey List
  3. Incomplete Survey

  5. Update Application
  6. About
  7. Logout

Dashboard: with the click on this button user will redirect to the dashboard from where you will start a new survey

Survey List:In the survey list you will get all the new survey list which are available.

Incomplete Surveys: In incomplete surveys you will get all of your stoped surveys here . And user can start from he felt the surve

Complete Survey: This is a main and important step where all the complete surveys stored you can sync the survey . In this you get two sync option

  1. sync all
  2. sync only unsynced surveys

with the sync all all the survey data will be synced to the server and with the sync only un- synced it will sync only unsync survey data.
